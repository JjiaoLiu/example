indexOf:返回value在该 array 中的索引值，如果value不存在 array中就返回-1。 indexOf(array, value, [isSorted])

lastIndexOf:返回value在该 array 中的从最后开始的索引值，如果value不存在 array中就返回-1。 _.lastIndexOf(array, value, [fromIndex])

sortedIndex:使用二分查找确定value在list中的位置序号，value按此序号插入能保持list原有的排序。 _.sortedIndex(list, value, [iteratee], [context])

findIndex:类似于_.indexOf，当predicate通过真检查时，返回第一个索引值；否则返回-1。 _.findIndex(array, predicate, [context])

findLastIndex:和_.findIndex类似，但反向迭代数组，当predicate通过真检查时，最接近末端的索引值将被返回。 _.findLastIndex(array, predicate, [context])
